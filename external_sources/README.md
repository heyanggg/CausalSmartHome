# external_sources

Keep this directory as the stable location for local upstream checkouts or symlinks.

Recommended server layout:

```bash
ln -s /home/heyang/projects/SmartGen external_sources/SmartGen
ln -s /home/heyang/projects/GCAD external_sources/GCAD
```

SmartGuard can also be linked for historical extension work, but it is not the main downstream AD pipeline in the corrected Stage 4 experiments.
